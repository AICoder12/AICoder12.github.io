Knowledge-Sharing Prompt Network for Continual Defect Segmentation

Abstract

Defect segmentation plays a vital role in industrial quality inspection, with deep learning methods becoming the mainstream solution. In practical production, defect data accumulates over time, and new defect classes can emerge at any time. However, when adapting to new classes, existing models tend to suffer from catastrophic forgetting of old classes, causing significant accuracy degradation. This work tackles the challenge and introduces the task of continual defect segmentation (CDS) to investigate how to continuously learn new defect classes without forgetting previously acquired class knowledge. To address this issue, we propose a Knowledge-Sharing Prompt Network (KSPNet), which integrates multi-task learning mechanism with the frozen network to enable retention and continual expansion of knowledge from incremental class training, and fine tunes complementary prompt embeddings for efficient adaptation to new classes. Specifically, to enable generalized prompt inference on the frozen network, a Knowledge-Balanced Low-Rank Adaptation (KB-LoRA) module is designed to incorporate multi-task low-rank matrices into image encoder, effectively balancing feature learning between new and old defect classes under multi-task loss constraints. Meantime, a Complementary Prompt Block (CPBlock) is designed to introduce task-invariant and task-specific prompts for adaptive learning, enabling the acquisition of complementary prompt information to guide continual defect segmentation. Extensive experiments conducted on the constructed CID-Seg and WD-Seg datasets have shown that the proposed method significantly outperforms existing methods and validates its superiority in the CDS task.
